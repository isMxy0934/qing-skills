# 输出字段完整说明

## 顶层结构

```json
{
  "code": "600519",
  "name": "贵州茅台",
  "market": "A股",
  "source": "akshare",
  "update_time": "2024-01-15 15:00:00",
  "klines": [...],
  "realtime": {...},
  "chip": {...}
}
```

| 字段 | 类型 | 说明 |
|-----|------|------|
| code | string | 股票代码 |
| name | string | 股票名称（来自实时行情，失败时=code） |
| market | string | 市场类型：A股/港股/美股/ETF |
| source | string | 数据源标识（固定为 "akshare"） |
| update_time | string | 数据更新时间 |

---

## klines（K线数据）

**数组格式**，每个元素代表一天：

```json
{
  "date": "2024-01-15",
  "open": 1688.50,
  "high": 1705.00,
  "low": 1680.00,
  "close": 1698.00,
  "volume": 12580000,
  "amount": 21348000000.0,
  "pct_chg": 0.56
}
```

| 字段 | 类型 | 说明 | 单位 |
|-----|------|------|------|
| date | string | 交易日期 | YYYY-MM-DD |
| open | float | 开盘价 | 元 |
| high | float | 最高价 | 元 |
| low | float | 最低价 | 元 |
| close | float | 收盘价 | 元 |
| volume | int | 成交量 | 股/手 |
| amount | float | 成交额 | 元 |
| pct_chg | float | 涨跌幅 | 百分比 |

**注意**：价格已前复权（qfq），消除了除权除息影响。

---

## realtime（实时行情）

**对象格式**，美股此字段为 `null`：

```json
{
  "name": "贵州茅台",
  "price": 1698.50,
  "change_pct": 0.56,
  "change": 9.50,
  "open": 1688.50,
  "high": 1705.00,
  "low": 1680.00,
  "volume": 12580000,
  "amount": 21348000000.0,
  "volume_ratio": 1.23,
  "turnover_rate": 0.68,
  "pe_ratio": 35.2,
  "pb_ratio": 8.5,
  "total_mv": 2130000000000.0,
  "circ_mv": 2130000000000.0,
  "amplitude": 1.48,
  "high_52w": 1850.00,
  "low_52w": 1450.00,
  "change_60d": 12.3
}
```

### 核心字段

| 字段 | 说明 | 关键阈值 |
|-----|------|---------|
| **volume_ratio** | 量比（今日成交量 / 5日均量） | >1.5 放量<br><0.7 缩量 |
| **turnover_rate** | 换手率（%） | >10% 高换手<br><1% 低活跃 |
| **amplitude** | 振幅（%）= (最高-最低)/昨收 | >7% 波动大 |

### 估值字段

| 字段 | 说明 | 参考标准 |
|-----|------|---------|
| **pe_ratio** | 市盈率（动态TTM） | <15 低估<br>15-30 合理<br>>50 高估 |
| **pb_ratio** | 市净率 | <1 破净<br>1-3 合理 |

### 市值字段

| 字段 | 说明 | 单位 |
|-----|------|------|
| **total_mv** | 总市值 | 元 |
| **circ_mv** | 流通市值 | 元 |

### 区间指标

| 字段 | 说明 |
|-----|------|
| **high_52w** | 52周最高价 |
| **low_52w** | 52周最低价 |
| **change_60d** | 60日涨跌幅（%） |

## chip（筹码分布）

**对象格式**，仅A股支持，其他市场为 `null`：

```json
{
  "date": "2024-01-15",
  "profit_ratio": 0.856,
  "avg_cost": 1620.0,
  "concentration_90": 0.125,
  "concentration_70": 0.083,
  "cost_90_low": 1520.0,
  "cost_90_high": 1750.0,
  "cost_70_low": 1580.0,
  "cost_70_high": 1680.0
}
```

### 获利指标

| 字段 | 说明 | 关键阈值 |
|-----|------|---------|
| **profit_ratio** | 获利比例（0~1） | >0.7 警惕回调<br><0.3 深套 |
| **avg_cost** | 平均成本价 | 当前价 > 均成本 = 大部分获利 |

### 集中度指标

| 字段 | 说明 | 关键阈值 |
|-----|------|---------|
| **concentration_90** | 90%筹码集中度（0~1） | <0.10 高度集中<br>>0.20 分散 |
| **concentration_70** | 70%筹码集中度（0~1） | <0.08 极度集中 |

**集中度计算**：包含70%/90%筹码的价格区间占股价的比例（0~1），越小越集中。

### 成本区间

| 字段 | 说明 | 用途 |
|-----|------|------|
| **cost_90_low** | 90%筹码下限价格 | 关键支撑位 |
| **cost_90_high** | 90%筹码上限价格 | 关键压力位 |
| **cost_70_low** | 70%筹码下限价格 | 主力成本下沿 |
| **cost_70_high** | 70%筹码上限价格 | 主力成本上沿 |

**应用**：当前价突破 `cost_90_high` 通常意味着 90% 筹码获利，容易诱发抛压。

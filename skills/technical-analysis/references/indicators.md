# 技术指标参考

## 评分标准速查

| 分数区间 | 信号 | 条件 |
|---------|------|------|
| 75-100 | 强烈买入 | 多头+高分 |
| 60-74 | 买入 | 弱多头以上+60分 |
| 45-59 | 持有 | 条件一般 |
| 30-44 | 观望 | 条件较差 |
| 0-29 | 卖出/强烈卖出 | 空头趋势 |

## 评分权重

| 维度 | 分值 | 说明 |
|------|------|------|
| 趋势 | 30分 | 多头排列得分高 |
| 乖离率 | 20分 | 接近MA5得分高 |
| 量能 | 15分 | 缩量回调得分高 |
| MACD | 15分 | 金叉和多头得分高 |
| 支撑 | 10分 | 获得均线支撑得分高 |
| RSI | 10分 | 超卖和强势得分高 |

## 趋势状态（7级）

| 状态 | 条件 | 强度 |
|------|------|------|
| 强势多头 | MA5>MA10>MA20 且间距扩大 | 90 |
| 多头排列 | MA5>MA10>MA20 | 75 |
| 弱势多头 | MA5>MA10 但 MA10≤MA20 | 55 |
| 盘整 | 均线缠绕 | 50 |
| 弱势空头 | MA5<MA10 但 MA10≥MA20 | 40 |
| 空头排列 | MA5<MA10<MA20 | 25 |
| 强势空头 | MA5<MA10<MA20 且间距扩大 | 10 |

## MACD 状态（7种）

| 状态 | 条件 | 信号 |
|------|------|------|
| 零轴上金叉 | DIF上穿DEA 且 DIF>0 | 最强买入 |
| 金叉 | DIF上穿DEA | 买入 |
| 上穿零轴 | DIF上穿零轴 | 趋势转强 |
| 多头 | DIF>DEA>0 | 持续上涨 |
| 空头 | DIF<DEA<0 | 持续下跌 |
| 下穿零轴 | DIF下穿零轴 | 趋势转弱 |
| 死叉 | DIF下穿DEA | 卖出 |

## RSI 状态（5种）

| 状态 | RSI(12)范围 | 信号 |
|------|-------------|------|
| 超买 | >70 | 短期回调风险 |
| 强势 | 60-70 | 多头力量充足 |
| 中性 | 40-60 | 震荡整理 |
| 弱势 | 30-40 | 关注反弹 |
| 超卖 | <30 | 反弹机会大 |

## 量能状态（5种）

| 状态 | 条件 | 含义 |
|------|------|------|
| 缩量回调 | 量比<0.7 且 下跌 | 洗盘特征（好） |
| 放量上涨 | 量比>1.5 且 上涨 | 多头力量强 |
| 量能正常 | 0.7-1.5 | 正常波动 |
| 缩量上涨 | 量比<0.7 且 上涨 | 动能不足 |
| 放量下跌 | 量比>1.5 且 下跌 | 注意风险 |

## 乖离率标准

| 乖离率(MA5) | 状态 | 操作建议 |
|-------------|------|----------|
| <-3% | 回踩买点 | 可积极介入 |
| -3%~0% | 安全 | 可介入 |
| 0%~2% | 安全 | 最佳买点区间 |
| 2%~5% | 警戒 | 可小仓介入 |
| >5% | 危险 | 严禁追高 |

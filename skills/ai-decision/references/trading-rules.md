# 交易理念与规则

## 核心交易理念

### 1. 严进策略（不追高）
- **绝对不追高**：乖离率 > 5% 时坚决不买入
- 乖离率 < 2%：最佳买点区间
- 乖离率 2-5%：可小仓介入
- 乖离率 > 5%：严禁追高

### 2. 趋势交易（顺势而为）
- **多头排列必须条件**：MA5 > MA10 > MA20
- 只做多头排列的股票
- 空头排列坚决不碰

### 3. 效率优先（筹码结构）
- 90%集中度 < 15% 表示筹码集中
- 获利比例 70-90% 时需警惕获利回吐

### 4. 买点偏好（回踩支撑）
- **最佳买点**：缩量回踩 MA5 获得支撑
- **次优买点**：回踩 MA10 获得支撑
- **观望情况**：跌破 MA20 时观望

## 风险排查重点

1. 减持公告（股东、高管减持）
2. 业绩预亏/大幅下滑
3. 监管处罚/立案调查
4. 行业政策利空
5. 大额解禁

## 检查清单模板

| 检查项 | 状态 | 说明 |
|--------|------|------|
| 多头排列 | YES/NO | MA5>MA10>MA20 |
| 乖离率<5% | YES/NO | 现价与MA5偏离度 |
| 量能配合 | YES/NO | 缩量回调优先 |
| 无重大利空 | YES/NO | 舆情检查 |
| 筹码健康 | YES/NO | 获利盘<70% |
| 买点位置 | YES/NO | MA支撑有效 |

## 仓位策略

| 评分 | 信号 | 建议仓位 |
|------|------|----------|
| 80+ | 强烈买入 | 50-70% |
| 60-79 | 买入 | 30-50% |
| 40-59 | 观望 | 不建仓 |
| <40 | 卖出 | 减仓/清仓 |

## 止损原则

- 跌破 MA20 止损
- 单票最大亏损 5%
- 单票仓位不超过总资金 20%
